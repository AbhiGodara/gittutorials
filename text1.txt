1.git status->check status
2.git add text1.txt->add file

4.git restore text1.txt->restore new changes
5.git commit -m "commit"->will commit
6.git remote add origin git@github.com:AbhiGodara/gittutorials.git->will add to temp like mode
7.git push -u origin main->will push with commit
